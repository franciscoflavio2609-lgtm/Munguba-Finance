# 🌳 Munguba Finance

App de educação e controle financeiro pessoal.

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Deploy no Vercel

1. Faça push para o GitHub
2. Importe o repositório no vercel.com
3. Clique em Deploy — pronto!

## Tecnologias
- React 18 + Vite
- Chart.js / react-chartjs-2
- CSS-in-JS (inline styles)
